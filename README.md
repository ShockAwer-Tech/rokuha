# What is futabawall?
futabawall is a translation of futaba-ng, which is a rewrite of the old futaba channel software.

# Instructions
1. Open futaba.php.

2. Edit lines 10-34 (optional, but it's recommended to change the admin password).

3. Upload the 3 php files.

You are done!
